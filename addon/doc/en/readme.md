# Plain View

NVDA add-on for piping terminal text into Notepad for easier screen-reader navigation, with quickjump-style bookmarks for jumping around the dumped content.

## Gestures

- **Open Plain View** — copy text from the current terminal, dump it to a temp file, and open it in Notepad.
- **Open Plain View with Claude Code attention jump** — same as above, then jump the cursor to the Claude Code "attention point" in the dumped text.

Gestures are unbound by default. Assign keys via NVDA's Input Gestures dialog under the "Plain View" category.
